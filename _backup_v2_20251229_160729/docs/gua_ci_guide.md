# 卦辭爻辭擴充指南

## 現況

目前已在 `divination_core.py` 中添加了卦辭爻辭功能，但只提供了 3 個範例卦：
- 乾為天
- 坤為地
- 水雷屯

## 如何擴充

在 `divination_core.py` 的 `GUA_CI` 字典中添加更多卦：

```python
GUA_CI = {
    '卦名': {
        'gua_ci': '卦辭原文',
        'yao_ci': {
            1: '初爻爻辭',
            2: '二爻爻辭',
            3: '三爻爻辭',
            4: '四爻爻辭',
            5: '五爻爻辭',
            6: '上爻爻辭'
        }
    },
    # ... 更多卦
}
```

## 參考來源

可以從以下來源獲取易經原文：
1. **維基文庫**：https://zh.wikisource.org/wiki/周易
2. **線上易經全文**：各大國學網站
3. **古籍數據庫**

## 優先級建議

建議優先添加常見的卦：
1. **八純卦**（8個）：乾、坤、震、巽、坎、離、艮、兌
2. **常用吉卦**：泰、臨、復、益等
3. **常用凶卦**：否、剝、困、蹇等
4. **其他 64 卦**：根據使用頻率逐步補充

## 自動化方案

如果需要批量添加，可以：
1. 找到包含 64 卦完整文本的 JSON/CSV 文件
2. 編寫腳本自動轉換格式
3. 或使用網路爬蟲從維基文庫抓取

## 當前功能

即使某些卦尚未添加卦辭爻辭，排盤功能仍然正常工作，只是在輸出中不會包含 `gua_ci` 和 `yao_ci` 字段。

AI 仍可根據卦象的六親、世應、動爻等信息進行解讀。
