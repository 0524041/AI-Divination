# 六爻排盤格式化輸出說明

## 概述

當六爻排盤結果傳入 AI 進行解讀時，會經過 `format_divination_result()` 函數格式化成易讀的文本格式。

## 格式化後的輸出範例

```
============================================================
起卦時間: 2025-12-29 14:30:00
八字: 乙巳 戊子 壬申 丁未
空亡: 戌亥
卦宮: 乾
本卦: 乾為天
變卦: 無變卦
卦類型: 本宮卦                    <-- ✅ 新增
============================================================
求測者性別: 男
占卜對象: 自己
============================================================
神煞: 驛馬:['寅'], 桃花:['酉']
------------------------------------------------------------
爻位   六神     六親     地支   五行   爻象   世應   動    變爻
------------------------------------------------------------
6爻    白虎    父母    戌   土   ⚊   世       
5爻    螣蛇    兄弟    申   金   ⚊           
4爻    勾陳    官鬼    午   火   ⚊           
3爻    朱雀    父母    辰   土   ⚊   應       
2爻    青龍    妻財    寅   木   ⚊

============================================================

------------------------------------------------------------

  九五：飛龍在天，利見大人。
  九四：或躍在淵，無咎。

  九二：見龍在田，利見大人。
  初九：潛龍，勿用。

```

## 包含的信息

### 基本信息區
1. **起卦時間**
2. **八字** - 年月日時的干支
3. **空亡** - 旬空的地支
4. **卦宮** - 所屬八宮
5. **本卦** - 本卦名稱
6. **變卦** - 變卦名稱（如果有動爻）
7. **卦類型** ✅ NEW - 本宮卦/一世卦/二世卦/三世卦/四世卦/五世卦/遊魂卦/歸魂卦

### 求測者信息區（可選）
- **性別** - 男/女
- **占卜對象** - 自己/父母/朋友/他人

### 神煞區
列出驛馬、桃花等神煞信息

### 六爻排盤表格
表格包含以下列：
- **爻位** - 1-6爻
- **六神** - 青龍、朱雀、勾陳、螣蛇、白虎、玄武
- **六親** - 父母、兄弟、官鬼、妻財、子孫
- **地支** - 子丑寅卯等
- **五行** - 金木水火土
- **爻象** - ⚊(陽) 或 ⚋(陰)
- **世應** - 世爻或應爻標記
- **動** - ○ 表示動爻
- **變爻** - 如果是動爻，顯示變出的六親、地支、五行

### 易經原文區 ✅ NEW（如果有的話）
- **卦辭** - 該卦的卦辭原文
- **爻辭** - 六個爻辭（從上爻到初爻排列）

## 新增字段的處理

### 卦類型
- **位置**：基本信息區，變卦下方
- **顯示**：始終顯示，即使無卦辭
- **意義**：
  - 本宮卦：最穩定
  - 遊魂卦：不安定、變動
  - 歸魂卦：回歸、結束

### 卦辭爻辭
- **位置**：六爻排盤表格下方獨立區塊
- **顯示**：僅當該卦有卦辭數據時顯示
- **格式**：
  - 卦辭單行顯示
  - 爻辭按上→下順序縮排顯示

## AI 使用方式

這個格式化後的文本會被插入到 system prompt 的 `{divination_result}` 位置，AI 會根據：

1. **卦類型** - 判斷事情的穩定性
2. **六爻排盤** - 分析旺衰、生克
3. **易經原文** - 印證卦象含義
4. **求測者信息** - 針對性解讀

給出專業的六爻解卦結果。

## 優勢

✅ **結構清晰** - 表格化呈現，易於 AI 理解
✅ **信息完整** - 包含所有斷卦所需元素
✅ **向後兼容** - 即使某些卦無卦辭，仍正常工作
✅ **可擴展** - 未來可添加更多字段（如伏神）
