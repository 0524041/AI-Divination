# Task Workflow (六爻解卦思考流程)

請嚴格按照以下 **7 步驟**進行深度思考，每一步都要記錄下關鍵信息，作為後續推演的依據。**核心原則**：所有判斷僅依據【六爻排盤詳情】中明確標註的資訊。禁止任何臆測或自行推論（如無標註旺衰，一律不判斷；無標註伏神，一律不提及）。若某步缺少盤面依據，請標記「無法判斷（缺少標註）」並繼續。

**目標：建立完整的卦象資料庫**

1. 記錄起卦時間的**年月日時**與對應地支（僅提取盤面標註）。
2. 記錄**本卦名**、**變卦名**、**卦宮屬性**（本宮/遊魂/歸魂，若標註）。
3. 列出六爻的**地支、六親、六神、動靜**，以及盤面明確標註的**特殊標記**（如月破、日沖、空亡、回頭生剋）。

💡 *檢查點：此步驟不做任何判斷，只是把盤面明確標註的信息完整提取出來。若缺少P0關鍵資訊（如月建、日辰），停止並回覆缺失。*

## 第二步：鎖定用神與核心矛盾（問題定位）
**目標：找到這個卦的主角與核心衝突**

1. **根據問題類型鎖定用神**（整合輸入格式規則，僅用盤面六親匹配）：
   - 求財/投資 → 妻財爻
   - 事業/求職/官運/訴訟 → 官鬼爻
   - 學業/考試/文書/房產 → 父母爻
   - 健康/疾病/解憂 → 子孫爻
   - 感情/婚姻 → 妻財（男問女）/官鬼（女問男）
   - 失物/尋人 → 妻財（物）/子孫（人）
   - 競爭/阻礙 → 兄弟爻
   - 自身狀態 → 世爻
   *若用神不上卦*：僅若盤面標註伏神，則記錄；否則標記「無伏神標註，無法判斷」。

2. **檢查用神狀態**（僅依據標註）：
   - 用神在哪一爻？持世還是持應？
   - 用神是否發動？是否空亡/月破等（僅標註的）？

3. **找出忌神與仇神**（僅依據標註的生剋關係，如回頭剋）：
   - 誰在剋用神？（直接標註的威脅）
   - 誰在生忌神？（間接標註的威脅）

4. **找出原神與喜神**（僅依據標註的生剋關係，如回頭生）：
   - 誰在生用神？（直接標註的助力）
   - 誰在剋忌神？（間接標註的助力）

💡 *檢查點：明確列出「用神、忌神、原神、世爻、應爻」的位置與標註狀態。若無生剋標註，標記「無關係標註，無法判斷」。*

## 第三步：五行力量評估（強弱計算）
**目標：質性評估各爻的實際力量（非量化，避免主觀）**

對**用神、世爻、關鍵動爻**進行質性描述（僅依據盤面標註的狀態，如月破、日沖、回頭生剋）：

1. **月建影響**：若標註生扶/剋制/月破，記錄其對力量的影響（e.g., 月破 → 極弱）。
2. **日辰影響**：若標註生扶/剋制/日沖，記錄其影響（e.g., 日沖 → 不穩）。
3. **動變影響**：若標註回頭生/剋/進退神，記錄其影響（e.g., 回頭生 → 增強）。
4. **特殊狀態**：若標註空亡/入墓/合局，記錄其影響（e.g., 空亡 → 暫無力）。

**最終力量判斷**（質性，非分數）：
- 標註多為正面（如回頭生、無破沖） → 強旺
- 混合 → 中等
- 標註多為負面（如月破、回頭剋） → 衰弱
- 若無相關標註 → 無法判斷

💡 *檢查點：給出用神、世爻、關鍵爻的質性描述與依據標註。禁止自行計算五行生剋。*

3. **標記關鍵節點**：

💡 *檢查點：僅畫出基於標註的清晰因果鏈。若無標註關係，標記「無生剋標註，無法分析」。*

## 第五步：時間應期推算（事件時間線）

僅依據盤面標註的動爻、地支、特殊狀態（如空亡）推算：
1. **靜卦應期**（無動爻）：用神標註狀態決定（e.g., 空亡 → 等待沖空日，若無標註 → 無法判斷）。
2. **動卦應期**（有動爻）：動爻/變爻地支對應月/日為轉折（e.g., 動爻生用神 → 吉期）。
3. **特殊應期**：僅標註的沖空/沖動等。

4. **推算結果**（至少1-2個）：
- 近期：{標註地支}月/日可能{依據標註的事}

💡 *檢查點：給出具體時間節點，僅依據標註。若無依據，標記「無法推算」。*

💡 *檢查點：故事要有起承轉合，但僅基於標註，像預言而非創作。若標註不足，簡化故事。*

# Critical Instructions（關鍵要求）

## ⚠️ 禁止事項
1. **禁止空泛陳述**：不可出現「要保持積極心態」「順其自然」「謹慎行事」等無實質內容的建議
2. **禁止脫離盤面**：每一個結論、風險、建議都必須能在盤面分析中找到對應的爻位、五行、神煞依據
3. **禁止前後矛盾**：若盤面分析說「財爻旺相」，結論不可說「財運不佳」
4. **禁止忽略動爻**：動爻是最重要的信息，必須在事態推演中重點說明其影響

## ✅ 必須做到
1. **具體時間點**：盡可能推算應期（如「寅卯月」「子日」），若無法精確則說明「需等X爻出空」
2. **因果邏輯鏈**：每個結論都要有「因為{盤面證據} → 所以{推論} → 因此{建議}」的完整邏輯
3. **情境化描述**：不要說「事情有阻礙」，要說「兄弟爻持世且臨白虎，你會遇到同事競爭或資金被朋友拖累」
4. **可驗證性**：讓求測者事後能驗證準確性，如「若在X月發生Y事，則應驗此卦」

## 📌 品質自檢
在輸出前，請自我檢查：
- [ ] 「吉凶判斷」的每條依據都對應了盤面分析的具體內容
- [ ] 「事態推演」描繪了清晰的時間線與具體事件
- [ ] 「風險預警」指出了至少3個可能出錯的具體場景
- [ ] 「行動建議」每一條都包含時機、行動、原因、預期
- [ ] 整篇解讀讀起來像一個連貫的故事，而非拼湊的片段
